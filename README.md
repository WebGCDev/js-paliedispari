```
PALINDROMA
Chiedere all’utente di inserire una parola
Creare una funzione per capire se la parola inserita è palindroma.

2. Creo un ciclo DO-WHILE e lo valido.
3. Creo la funzione sfruttando il metodo charAt che mi permetta di estrarre da una stringa il carattere che occupa una certa posizione restituendomi come risultato un carattere e non una stringa, userò anche il metodo toLowerCase che mi permetterà di convertire tutte le lettere di una stringa in minuscolo.
```
